# Diabetes Patient Readmission Analysis

## Overview

This project analyses diabetes patient readmission patterns using healthcare data to identify factors associated with hospital readmissions. The objective was to transform raw patient data into meaningful insights through data cleaning, feature engineering, exploratory data analysis (EDA), and visualisation.

The analysis investigates relationships between patient demographics, admission types, medication usage, length of hospital stay, and readmission outcomes to support data-driven healthcare decision-making.

## Objectives

* Explore diabetes patient readmission trends.
* Identify factors that influence hospital readmissions.
* Analyse the impact of age, gender, race, and admission type on readmission outcomes.
* Investigate medication usage and length of stay patterns.
* Generate actionable insights through visual analytics.

## Methodology

### Data Preparation

* Cleaned and transformed healthcare datasets.
* Handled missing and inconsistent values.
* Created new analytical features.
* Standardised categorical variables for analysis.

### Exploratory Data Analysis

* Readmission distribution analysis.
* Admission type analysis.
* Age and gender comparisons.
* Length of hospital stay analysis.
* Medication usage analysis.
* Medical speciality analysis.

### Data Visualisation

Visualisations were created to highlight key trends and relationships within the data, making insights easier to interpret and communicate.
